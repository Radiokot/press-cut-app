# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed

- Stamps now appear in the phone gallery faster
- New stamps no longer change their date in the phone gallery when moved between collections

## [1.4.1] - 2026-06-23

### Fixed

- Crashing on Android 9 and older when no permissions are granted

### Changed

- Made the floating messages prettier

## [1.4.0] - 2026-06-13

### Added

- Ability to share individual stamps as posters
- A landscape version of the stamp screen

## [1.3.0] - 2026-06-05

### Added

- Ability to send individual stamps as images
- Dark theme

## [1.2.0] - 2026-05-30

### Added

- Support for more OneStamp shapes
- Ability to send stamp envelopes, to both PressCut and OneStamp on iOS

## [1.1.0] - 2026-05-25

### Added

- Temperature adjustment
- Glitch effect
- Ability to open envelopes sent from OneStamp on iOS

### Changed

- Made the Vibrance adjustment stronger

## [1.0.1] - 2026-05-15

### Fixed

- Incorrect start destination in the directory access dialog on some devices
